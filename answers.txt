What is a gitignore file and what should it include for a Java project? 
gitignore file is a text file in the git repository that tells git not to track files and folders which you do not want in your master.
In a java project gitignore usually includes .jar,.class,.war etc. also it includes the files which are generated by the ide which is 
used in the project.

What does fast-forward in git mean? 
fast forward merge is done when there is direct linear path from source to target branch. In this git moves the source branch pointer to 
target branch pointer without creating extra merge commit.

Why should you have different branches? 
creating branches allows each member of the team to deviate from the main branch and do their work on other branches or fix any bugs.
It helps to merge the developments once the project is completed.

In a project how many remote Git repositories do you usually have? 
one project will have one remote git repository and any number of local repositories
as each member will clone git repository to their local repositories.
